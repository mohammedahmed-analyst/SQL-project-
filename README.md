📚 Online Bookstore SQL Project:-

This project presents a structured implementation of an Online Bookstore database system using SQL. It focuses on designing a relational database, managing data, and performing analytical queries to extract meaningful insights.

The database consists of three main entities: Books, Customers, and Orders, connected through primary and foreign key relationships to maintain data integrity and represent real-world interactions between users and products.

Data is imported from external CSV files into the database, enabling realistic data handling and storage. The project includes a wide range of SQL queries, starting from basic data retrieval to advanced operations using joins, aggregations, grouping, and filtering.

Through these queries, the system analyzes key aspects such as book availability, customer activity, order details, sales trends, and total revenue. It also identifies patterns like most frequently ordered books, high-value customers, and stock status after order fulfillment.

Overall, this project demonstrates practical usage of SQL for database design, data manipulation, and business-oriented data analysis within a bookstore management system.
